## Pedagogical Framework

### Core Methodology
- **Communicative Language Teaching (CLT)** - Real-world communication focus
- **Task-Based Language Learning (TBLL)** - Learning through meaningful tasks  
- **Cultural Integration** - Ubuntu/Unhu principles throughout
- **Dual Pathways** - Age-appropriate content delivery

### Learning Progression
1. **Level 1 (3 months)** - Survival Shona, 1,000-1,500 words
2. **Level 2 (4 months)** - Narrative skills, 3,000-5,000 words  
3. **Level 3 (5 months)** - Cultural fluency, 5,000+ words
4. **Level 4 (6 months)** - Advanced mastery and rhetoric
